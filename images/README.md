# Project Visualizations

This folder contains the main charts generated during the analysis and used in the main README to highlight key insights.

## Visuals included

### Customer Clustering – Cycle Time vs Idle Time
`cluster_2.png`  
Scatter plot showing customer clusters based on average cycle time and idle time.

### Courier Performance – Top 15 Couriers
`corriere.png`  
Bar chart ranking the top 15 couriers with the lowest average idle time.

### Customer Cluster Profiles
`cluster dei clienti.png`  
Comparison of average features (missions count, cycle time, service time, idle time) across customer clusters.

These visualizations are intended to provide a quick, high-level understanding of operational patterns and customer segmentation results.
